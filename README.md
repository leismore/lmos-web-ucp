# lmos-web-ucp

An under-construction web page from [Leismore OpenSource for Web (LMOS for Web)](https://web.lmos.leismore.org) project.

## Motivation

To build a generic under-construction web page template for everyone.

## Customisation

Update `[your_content]` in the `index.html` file.

## Deployment

Upload the `public` folder to a web server.
